# 重新启动说明

## 问题原因

你看到的界面还有"➕ 添加命令"等按钮,是因为 Python 缓存了旧版本的代码(.pyc 文件)。

## 解决方法

我已经清理了缓存,现在请按以下步骤重新启动:

### 方法 1: 使用启动脚本(推荐)

```bash
./run.sh
```

### 方法 2: 直接运行

```bash
python3 main.py
```

### 方法 3: 完全清理后启动

```bash
# 清理所有缓存
find . -name "*.pyc" -delete
find . -name "__pycache__" -type d -exec rm -rf {} +

# 启动程序
./run.sh
```

## 清理的缓存文件

已删除:
- `__pycache__/`
- `src/__pycache__/`
- `*.pyc` 文件

## 新界面预览

重新启动后,你应该看到:

```
┌─────────────────────────────────────┐
│ 命令列表                             │
│ 每行一条命令,编辑后自动保存          │
├─────────────────────────────────────┤
│                                     │
│  claude --dangerously-skip-permissions 诊断
│                                     │
├─────────────────────────────────────┤
│ [▶ 执行] [📂 查看结果]               │
└─────────────────────────────────────┘
```

**只有两个按钮:**
- ▶ 执行
- 📂 查看结果

**不再有:**
- ❌ 每条命令旁的删除按钮
- ▶ 每条命令旁的执行按钮
- ➕ 添加命令按钮

## 验证

重启后,检查界面是否符合以下要求:

- ✅ 命令列表区域只有一个文本框
- ✅ 底部只有 2 个按钮
- ✅ 没有"添加命令"按钮
- ✅ 命令旁没有单独的执行/删除按钮

如果还是看到旧界面,请:
1. 完全关闭应用程序
2. 运行 `rm -rf __pycache__ src/__pycache__`
3. 重新启动
