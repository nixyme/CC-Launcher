# English Screenshots Guide

## Required Screenshots for README_EN.md

The following screenshots need to be captured with the English interface:

### 1. Main Interface (main-ui-en.png)
- Switch language to English in settings
- Show the main interface with project list on the left and details on the right
- Recommended size: 800px width
- Save to: `docs/screenshots/main-ui-en.png`

### 2. New Project Modal (new-project-en.png)
- Click "New Project" button
- Show the modal with empty fields
- Recommended size: 800px width
- Save to: `docs/screenshots/new-project-en.png`

### 3. Command Execution (command-run-en.png)
- Select a project with commands
- Show the command panel with Run buttons
- Recommended size: 800px width
- Save to: `docs/screenshots/command-run-en.png`

### 4. Multi-Language Switching (i18n-en.png)
- Open settings modal
- Show the language dropdown with options
- Recommended size: 800px width
- Save to: `docs/screenshots/i18n-en.png`

## How to Capture

1. Run the app: `npm start`
2. Switch to English: Settings → Language → English
3. Use macOS Screenshot tool (⌘+Shift+4) or any screenshot tool
4. Crop to ~800px width for consistency
5. Save to `docs/screenshots/` with the names above
