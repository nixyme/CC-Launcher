# 快速开始指南

## 🚀 5分钟上手

### 第一步: 启动程序

```bash
./run.sh
```

或者

```bash
python3 main.py
```

### 第二步: 添加你的第一个项目

1. 点击左侧的 **"添加项目"** 按钮
2. 填写信息:
   - 项目名称: `我的测试项目`
   - 项目路径: 点击"浏览"选择你的 Claude Code 项目文件夹
   - 默认指令: 输入你想让 Claude 执行的任务,例如:
     ```
     帮我分析这个项目的代码结构
     ```
   - 结果路径: 选择你想查看结果的文件夹(可以是项目文件夹本身)
3. 点击 **"添加"**

### 第三步: 运行项目

1. 在左侧项目列表中点击选择你刚添加的项目
2. 右侧会显示项目详情和默认指令
3. 如果需要,可以修改指令内容
4. 点击 **"启动项目"** 按钮
5. 🎉 一个新的终端窗口会自动打开并开始执行 Claude Code!

### 第四步: 查看结果

运行完成后,点击 **"查看结果"** 按钮,文件管理器会自动打开结果文件夹。

## 💡 使用技巧

### 技巧1: 设置有用的默认指令

好的默认指令示例:
- `分析代码结构并生成文档`
- `检查代码中的潜在问题并提供修复建议`
- `优化代码性能`
- `添加单元测试`

### 技巧2: 管理多个项目

你可以添加多个项目,例如:
- `前端项目 - React`
- `后端项目 - Python API`
- `数据分析项目 - Jupyter`

每个项目都有自己的默认指令和结果路径,一键切换!

### 技巧3: 快速修改指令

选择项目后,你可以:
- 在默认指令基础上补充内容
- 完全重写指令
- 点击"重置为默认指令"恢复

### 技巧4: 设置结果路径

结果路径建议设置为:
- 项目的输出目录
- 项目根目录
- 统一的结果汇总文件夹

这样可以快速查看 Claude Code 的输出结果。

## ⚡ 常用操作

### 批量管理项目
1. 添加所有需要的项目
2. 为每个项目设置有意义的名称和默认指令
3. 随时选择并启动

### 快速切换项目
在项目列表中点击不同项目,即可快速切换。

### 保存配置
所有项目信息会自动保存在 `data/projects.json` 中,无需手动保存。

## 🎯 最佳实践

1. **明确的项目名称**: 使用清晰的名称,如"前端-Dashboard"而不是"项目1"
2. **有用的默认指令**: 设置常用的任务作为默认指令
3. **合理的结果路径**: 设置便于查看的结果目录
4. **定期清理**: 删除不再使用的项目
