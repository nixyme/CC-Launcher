# 综述
通过这个工具，可以把分布在不同目录下的claude code subagents项目，用通用的方法直接执行

# 功能描述
1.添加项目。选择本地项目的文件夹，然后给项目起个名字。设置一个{默认指令}和一个查看结果的文件夹。
2.添加后可以修改路径、名字、默认指令、查看结果路径。
3.运行项目。选中一个已添加的项目。自动加载{默认指令}，可以修改加载后的内容（可以改也可以继续补充），完成后点击“启动”，这个经过编辑的内容就是{指令}。然后启动一个新的终端，跳转到对应路径后，执行：claude --dangerously-skip-permissions {指令}
4.每个项目都有一个查看结果的按钮，点击自动用finder打开文件夹
