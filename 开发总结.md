### 技术栈
- **Python 3.8+**: 核心语言
- **Tkinter**: GUI 框架(内置,无需安装)
- **JSON**: 数据存储格式
- **subprocess**: 命令执行
- **platform**: 跨平台检测

### 模块设计

#### 1. project_manager.py - 数据管理层
- `ProjectManager` 类
- CRUD 操作封装
- JSON 文件读写
- 数据验证和异常处理
- 使用 UUID 作为项目唯一标识

#### 2. command_executor.py - 命令执行层
- `CommandExecutor` 类
- 跨平台终端命令执行
  - macOS: AppleScript + Terminal.app
  - Linux: gnome-terminal/xterm/konsole/xfce4-terminal
  - Windows: cmd.exe
- 跨平台文件夹打开
  - macOS: `open`
  - Linux: `xdg-open`
  - Windows: `explorer`

#### 3. gui.py - 用户界面层
- `ProjectLauncherGUI` 类
- 双栏布局(PanedWindow)
- 项目列表(Treeview)
- 详情编辑区(Entry + ScrolledText)
- 添加项目对话框(Toplevel)
- 事件绑定和状态管理

## 使用流程

1. **启动程序**: `./run.sh` 或 `python3 main.py`
2. **添加项目**: 点击"添加项目",填写信息
3. **选择项目**: 在列表中点击选择
4. **编辑指令**: 在右侧修改默认指令
5. **启动项目**: 点击"启动项目"按钮
6. **查看结果**: 点击"查看结果"按钮
